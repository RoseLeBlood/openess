31 **OpenESS** is a network-capable sound server libary mainly for embedded systems and other operatins systems.
32 OpenESS is free and open-source software, and is licensed under the terms of the GNU Lesser General Public License.
33 
34 ## Getting Started
35 
36 ### Setting Up Development Environment
37 
38 #### PlatformIO
39 
40 OpenESS is made for use with [platformio](http://platformio.org/), an advanced ecosystem for microcontroller
41 development. To get started with coding esphomelib applications, you first have to
42 [install the atom-based platformio IDE](http://platformio.org/platformio-ide) or for advanced users,
43 [install the command line version of platformio](http://docs.platformio.org/en/latest/installation.html).
44 
45 Then create a new project for an [ESP32-based board](http://docs.platformio.org/en/latest/platforms/espressif32.html#boards)
46 (for example, `nodemcu-32s`). Then open up the newly created `platformio.ini` file and insert
47 
48 ```ini
49 ; ...
50 platform = espressif32
51 board = nodemcu-32s
52 framework = esp-idf
53 lib_deps = openess
54 ```
55 Finally, create a new source file in the `src/` folder (for example `main.c`) and start coding with openess.

110 ## Current Features (version 0.2-1)
111 
112 * Powerful core that allows for easy to port
113 * Automatic WiFi handling (reconnects, etc.)
114 * Powerful socket abscrations layer (SAL)
115 * Easy to use platform configuration
116 * Semaphore, task and ringbuffer handling on various platform
117 * generic backends: udp, uart and i2s
118
