ess_socket_fam_t ess_get_address_family ( const char *  hostname)

Look up which address families a host supports.

If you want to send a datagram to a host but you don't know if it supports IPv4 or IPv6, use this function.

Parameters
hostnameThe hostname of the host you want to look up.
Return values
ESS_SOCKET_FAMILY_IP4Host supports only IP4
ESS_SOCKET_FAMILY_IP6Host supports IP6 and IP4
